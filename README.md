# 紫罗兰永恒花园主题网站

一个基于 HTML、CSS、JavaScript 的静态网站，展示《紫罗兰永恒花园》的剧情、人物、美图等内容，并包含用户注册/登录功能。


## 📋 项目简介
本项目是《紫罗兰永恒花园》的粉丝主题网站，旨在为爱好者提供作品相关的剧情介绍、角色信息、图片欣赏等内容，同时通过简单的注册/登录功能增强用户互动。


## ✨ 功能说明
1. **首页**：展示作品标题、作者信息及相关视频  
2. **剧情简介**：介绍动画的背景故事和核心情节  
3. **主要人物**：展示角色信息及配图  
4. **美图欣赏**：包含角色图片及交互效果（悬停动画等）  
5. **用户功能**：注册/登录表单（含前端验证）  


## 🚀 如何运行
1. 克隆或下载本项目到本地  
2. 直接打开 `index.html` 文件（无需依赖环境，静态页面可直接运行）  
3. 推荐使用现代浏览器（Chrome、Edge 等）访问
